# CinePhile — OMDb Client Hub

A clean, modern movie and TV discovery app for searching titles, filtering by type and year, and building a personal favorites list.

## Features
- Search movies, TV series, and episodes
- Filter by content type (All, Movie, Series, Episode) and release year
- Curated blockbuster highlights on load
- Sort results (e.g. Year: Newest)
- Save titles to Favorites
- Fully responsive design

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Run Locally
No installation or build step required.

1. Clone or download this repository
2. Copy `config.example.js` to `config.js` and add your free OMDb API key ([get one here](https://www.omdbapi.com/apikey.aspx))
3. Open `index.html` directly in your browser, or serve it with a local server (e.g. Live Server)
